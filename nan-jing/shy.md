# 1**关于高考后的选择**



**为什么会选择这个城市/学校/专业**

就学医而言，进不去老八校的同学一般会把目光放在首都医科大学、南京医科大学、南方医科大学这三所较强的医科类院校上，选择南医的原因首先是其地理位置，若在本省发展，江苏的医疗条件在长三角地带还算排的上号，若想进一步发展，向西可跳武汉同济协和两大医院，向东可跳上海复旦上交的顶尖医院。江苏的竞争压力相较于北京和广东而言会小一些，不至于被协和北医中大三所医学院压着打。

关于学校，南医是个四非学校，对985、211有执念的同学先要知道，对于福建招生而言，招生规模在2016年之后有所缩减，省排名大约在3000左右报临床专业比较稳，今年南医大概率能拿到双一流，分数线方面可能会涨，没涨之前报了就赚，另外附属医院的优势和科研的优势南医都没什么问题。

至于专业，我当时是调剂上的，和第一志愿差了两分，现在看来这是个好结果，我稍后介绍。

**当时有什么样的考量，那些考量现在觉得是可取的么**

南京医科大学这所学校说实话不推荐和我同样分数段的同学选择，向上可试试老八校，向下可以考虑福医大的长学制，不用怕浪费分数，学医选择省里最好的医学院基本不会有错误。南医的尴尬是得不到前两者的资源（八年制的资源比不上老八校五年制，五年制的资源比不上福医大八年制），并且八年制对于自己未来的提升是有局限性的。

**针对自己的学校专业，对学弟学妹们的志愿和专业选择有什么建议，是不是该选择这个专业，应该怎么选好适合自己的专业，需要考量哪些方面**

谈到专业，医学主要分为临床医学、基础医学、口腔医学和预防医学，分别对应临床科室、医学科研、头颈颌面口腔牙医学和公共卫生预防医学。

我的第一志愿是口腔，当时不了解以为口腔就是牙医，其实差得很远，口腔一个班30个人，只有1-2人可以从事口腔正畸学（也就是传统认知的牙医），其他的小类还有颌面外科，头颈外科等等非牙类专业。

基础医学方面，南医有一个生殖医学国家重点实验室和一个神经药理学省级重点实验室，这两个实验室我参与了三年，体验都还不错。

预防医学是南医的顶尖专业（本科分数不高），评级大概是全国第一了，领头人有沈洪兵院士、胡志兵教授做的流行病学研究，能进他们的实验室就很有发展了。

至于临床医学在本科没有细分科室，单独划分出来的只有儿科学、影像医学，这两者在临床医学的基础上需要额外学本专业知识。

关于影像学，临床阶段大致分为四类，影像诊断从事关于平片、CT、MRI的诊断工作、超声诊断从事关乎超声医学的操作和诊断工作，核医学从事分子靶向放射治疗，介入医学从事临床介入手术治疗工作。除了介入科是有射线危害，其余三科室都不吃线。进可发展介入医学上台手术并且收入不菲（相当于心内、神内等科室）、退可至办公室做诊断工作，不需要接触患者，非常安全，并且休息的时间也能得到保障，想稍微接触患者又不想做手术可以去超声诊断，喜欢闲着混日子可以去核医学，并且竞争相对于大临床要好很多。五年制的学制又不影响研究生去其他的学校提升，医学对于第一学历倒是没有那么看重，毕竟专科培养是从研究生开始的。

另外南医也有护理学、生物医学工程、英语等专业，就不细说了。



# 2**关于大学生活**



**在进入学校/专业学习之后，大致的生活和学习内容是怎样的，专业和学校特色是什么，有什么样的学习建议**

一般的医学生大学生活比较单一，就是学习。我不推荐这么做，希望大家多考虑自身情况找自己的路。

学习我就不说了，人命关天不能学差，最低保证GPA三分以上，专攻学习的保证在四分以上。

校园生活可以选择自己感兴趣的做，我选择了做设计和摄像，不要觉得医学生在这方面不能发展，医学方向加上这些东西是有发展的，总之多多积累自己。

在比赛方面也可以玩，互联网+、两个挑战杯、计算机设计大赛，这四个比赛是我比较推荐的，对未来的保研和创业都很有帮助。

在科研方面，起步可以去实验室做课题，之后没时间可以自己做meta分析和写综述类文章。

本科阶段的医学并不是专科发展而是全面发展，这样下来，学习、科研、技能、比赛都能做到一定成绩，对将来的选择很有好处。



# 3**关于将来的选择**



**最终选择保研/考研/留学/就业的考虑是什么，应该怎么考虑适合自己的道路**

关于留学，我非常推荐长学制的同学考虑，在没有考研压力、家庭条件又允许的情况下，完全可以在大一就开始准备出国行医。赴美行医的USMLE考试考了绝对不亏，Matching阶段运气不好没找到医院出不去也不需要担心没有研究生读，Matching上了就是聚聚。

保研的方法并不只有一种，五年制的同学在大二就要对自己评估，专攻学习的GPA能否到4以上，搞得比较花的同学就比较有趣，之前说的比赛都有加分，实验室混得好也有一两篇文章加加分，按照最新保研政策，想加满20分不是很难，这样对GPA的要求几乎等于毕业要求了。

不管能否保研，想当医生一定要准备考研，这样就算九月没保成也不会太狼狈，考研从大四上学期结束的寒假开始准备，怎么准备，去看考研经验比听我这个没考的更有意义。

如果想当医生，本科毕业就业比较困难，可能得到县级的医院了。当然学医也不一定要当医生，我之前说的额外技能的发展也可以帮到很大的忙，我业余做设计，稿子经常上丁香园这样的平台，之前有收到过他们的邀请，业余学摄像，之前在学校的党委宣传部工作，他们也接受我毕业留校任职，所以路子真的非常多，读书很重要但不一定最重要。



# 4**对于选择所需要的准备**



**什么时候决定保研/考研/留学/就业的**

虽然是五年制，但最好在大二的时候就要对自己有个定位了。分够不够，奖够不够，钱够不够，学力够不够。

**大概的时间节点有哪些，这些节点前都需要做哪些准备**

留学在大一先学好英语，大二开始准备，拿USMLE举例大概分为四个阶段，需要两到三年的时间准备，费用在二十万刀左右，不过考上了就业后一年就可回本。

保研凭分数或者比赛都得自己衡量，找对路就没啥好说，能不能保心里都清楚。

考研从大四的寒假开始，准备好苦战一年。

不推荐就业。

**过程中有哪些需要特别注意的细节，或者需要着重准备的方向**

打铁还需自身硬，不管哪条路，自身发展是最重要也是最关键的。

**最终选择现在的公司/学校/专业/城市的原因和考量**

****还在读，医学的选择包括城市、学校、医院、科室、导师，比较麻烦。个人认为导师和医院最重要。在中部地区的影像，华科附属同济医院、东大附属中大医院、南医附属第一医院，复旦附属中山医院，上交附属瑞金医院算是龙头了。

**给希望走这条道路的学弟学妹的建议**

最后，我不推荐大家用自己的兴趣作为专业选择的参考，把兴趣变为职业是十分奢侈的行为，搞不好你就会丢了一个兴趣并多了一份讨厌的职业。衡量一下什么职业最适合自己是最关键的，我只推荐三种人学医，一是心地善良用爱发电的，二是家里有钱想找个稳定行业的，三是适合学习的。读书、高考、读研、留学，这些转折确实像大家说的很关键，但它一定不是最关键的，在大方向正确的前提下，觉得对就去做，觉得开心就做好，然后不断迎来属于你们自己的闪光时刻。
